## üì¶ **Installation & Setup**

Follow these steps to run this project locally:

### **Prerequisites**
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)
- Local server environment (XAMPP, Live Server, etc.)

### **Installation Steps**
1. **Clone the repository**
   ```bash
   git clone https://github.com/anoopsinghji/E-learning-website-with-gamifications-and-AIChatbot.git
   ```

2. **Navigate to project directory**
   ```bash
   cd E-learning-website-with-gamifications-and-AIChatbot
   ```

3. **Open in your code editor**
   ```bash
   code .
   ```

4. **Run the project**
   - Option 1: Open `index.html` directly in your browser
   - Option 2: Use a local server (recommended)
     ```bash
     # If you have Live Server extension in VS Code
     # Right-click on index.html and select "Open with Live Server"
     ```

### **API Configuration**
1. **YouTube API Setup** (Optional for local development)
   - Get YouTube Data API key from [Google Cloud Console](https://console.cloud.google.com/)
   - Replace API key in `js/config.js`
   ```javascript
   const YOUTUBE_API_KEY = 'your-api-key-here';
   ```

2. **Botpress Configuration** (Optional)
   - Create account at [Botpress](https://botpress.com/)
   - Follow integration guide in documentation
